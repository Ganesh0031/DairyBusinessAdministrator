# 🥛 Dairy Business Administrator System

An enterprise-level Dairy Business Management System developed to digitize and automate daily dairy operations. The system provides a complete solution for managing farmers, milk collection, customer sales, accounting, invoices, employee management, and administrative tasks through a centralized platform.

The application consists of three major components:

- 🔹 Spring Boot REST API (Backend)
- 🔹 Flutter Mobile Application
- 🔹 React.js Admin Panel

---

# 🚀 Project Overview

The Dairy Business Administrator System helps dairy owners efficiently manage milk procurement, customer sales, financial transactions, rate calculations, invoices, employee operations, and dairy administration.

The system minimizes manual work by automating daily operations while maintaining complete transaction history and financial records.

# 🔑 Business Rules

- Every farmer has a unique account.
- Every customer has a separate ledger.
- Milk rates are automatically calculated based on FAT & CLR values.
- Morning and evening collections are maintained separately.
- Farmer payments are generated based on collected milk.
- Customer outstanding balances are automatically updated.
- Every transaction is stored for audit purposes.
- Invoice generation is automatic after payment processing.
